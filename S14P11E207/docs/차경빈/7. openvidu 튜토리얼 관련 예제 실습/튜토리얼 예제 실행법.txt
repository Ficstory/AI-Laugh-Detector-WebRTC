실행법
https://docs.openvidu.io/en/stable/tutorials/openvidu-hello-world/

1. git clone https://github.com/OpenVidu/openvidu-tutorials/tree/v2
cd openvidu-tutorials/openvidu-basic-java
mvn spring-boot:run
2. docker run -p 4443:4443 --rm -e OPENVIDU_SECRET=MY_SECRET openvidu/openvidu-dev:2.32.1
3. npm install --location=global http-server
http-server openvidu-hello-world/web
