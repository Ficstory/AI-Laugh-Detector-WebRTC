## 안건별 논의 내용

### 안건 1: MVP 수정

**논의 배경**

설문조사 결과 "얼굴 노출이 부담된다"는 응답이 다수 확인됨

**논의 내용**

- 랜덤 방 + 친구/지인 매칭 병행
- 필터 effect (우선순위 낮음)
- 모자이크 기능 (우선순위 낮음)
- 친구 매칭을 메인으로, 승리 누적 시 경쟁전 해금
- 캡처/녹화 방지 (우선순위 높으나 MVP는 아님)
- 악성 유저 방지를 위한 로그인 필수
- 이탈 유저 처리: 새로운 상대 재매칭으로 리텐션 유지
- 악성 유저 포인트 알고리즘 (신고, 평소 행위 기반)

**결정 사항 (Decision)**

✅ MVP 기능 재확정
- 랜덤 방 + 친구/지인 매칭
- 로그인 필수
- 준비 버튼 (READY)

✅ 우선순위 조정
- 방 기능 먼저 → 랜덤매칭 추가
- 방 간 이동 기능 검토

**결정 근거**

설문 응답 168개 분석 결과, 얼굴 노출 부담을 해소하기 위해 친구/지인 매칭을 메인으로 전환
