## 안건별 논의 내용

### 안건 1: 아이디어 피드백

**논의 배경**

웃음참기 배틀 아이디어에 대한 컨설턴트 검증 필요

**논의 내용**

- 실제로 쓸 법한 아이디어인지 검증 필요
- "토이 프로젝트" 느낌이 강하다는 우려
- 창의성, 기술성, 실용성 세 가지 관점에서 평가 필요
- 면접장에서 어떻게 발표할지 고민 필요

**결정 사항 (Decision)**

✅ 기술적 차별점 강화 필요
✅ 1월 13일까지 기획 확정
✅ 프로토타입 빠른 제작 권장

**결정 근거**

단순 유희 서비스가 아닌 기술력을 보여줄 수 있는 프로젝트로 방향 전환 필요

### 안건 2: 기술적 검토 사항

**논의 배경**

아이디어 실현을 위한 기술적 타당성 검토

**논의 내용**

- AI 모델 고도화 또는 다른 기술 디벨롭 필요
- 모델이 얼마나 정확하게 판단할 수 있는지 검증 필요
- 웃음 플랫폼의 기술적 측면도 중요하게 어필 필요

**결정 사항 (Decision)**

✅ 실사용 테스트를 위한 프로토타입 제작 우선
✅ 과거 개그 트렌드 변화 원인 조사

**결정 근거**

기술 스택 변경도 가능하므로 명세서에 과도하게 집착하지 않기로 함

### 안건 3: 향후 액션 아이템

**논의 배경**

컨설턴트 피드백 반영을 위한 구체적 액션 플랜 수립

**논의 내용**

- 서비스명 및 컨셉 정하기
- 실사용자 니즈 검증 방법 논의

**결정 사항 (Decision)**

✅ 설문조사 설계 및 배포 진행
✅ 1월 13일까지 기획 확정 목표

**결정 근거**

데이터 기반 의사결정을 위해 예비 사용자 의견 수집 필요
