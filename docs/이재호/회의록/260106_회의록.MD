## 안건별 논의 내용

### 안건 1: 역할 분담

**논의 배경**

7인 팀 구성에 따른 각 팀원의 역할과 담당 포지션을 명확히 정의한다.

**논의 내용**

- 백엔드/프론트엔드 희망 포지션 조율
- 일부 팀원의 크로스 학습 희망 사항 반영
- 공모전 참여 인원 별도 편성

**결정 사항 (Decision)**

✅ 역할 분담 확정
- **유준호**: 백엔드 (프론트 소통 지원)
- **차경빈**: 백엔드
- **이재호**: PM, 프론트엔드
- **김승철**: 프론트엔드 (백엔드 학습 희망)
- **오언서**: 백엔드, 클라우드 (공모전 참여)
- **박세홍**: 프론트엔드 (백엔드 학습 희망)
- **양한빈**: ML Ops (공모전 참여)

**결정 근거**

각 팀원의 희망 포지션과 역량을 고려하여 배분. 크로스 학습 희망자는 메인 역할 수행 후 여유 시 타 파트 참여 가능하도록 유연하게 구성.

### 안건 2: 공모전 참여

**논의 배경**

딥페이크 AI 판별 모델 공모전(마감: 1월 말)에 일부 팀원이 참여 희망

**논의 내용**

- 공모전 모델을 본 프로젝트에 연동 가능성 검토
- FastAPI로 연동 시 발표일 전 기능 추가 가능

**결정 사항 (Decision)**

✅ 양한빈, 오언서 + 1명 공모전 참여
✅ 딥페이크 판별 기능을 본 프로젝트에 FastAPI로 연동 예정

**결정 근거**

공모전 결과물을 프로젝트에 재활용하여 시너지 창출 가능
